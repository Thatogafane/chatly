# Chatly

Chatly is a chat app example using ably golang sdk

## How to run the application

Step 1: First create a .env file

Step 2: Add your ably api key and name it ABLY_API_KEY

Step 3: Type "go run . <Any Name You Like>" in your terminal

Step 4: Open a new terminal and repeat step 3

Step 5: Write any message you like and view the message pop up on the other terminal
